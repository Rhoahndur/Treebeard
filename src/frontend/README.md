# TreeBeard Frontend

React + TypeScript frontend for the AI Energy Plan Recommendation Agent.

## Structure

```
frontend/
├── src/
│   ├── components/    # React components
│   ├── pages/         # Page components
│   ├── hooks/         # Custom React hooks
│   ├── utils/         # Utility functions
│   ├── types/         # TypeScript type definitions
│   ├── api/           # API client
│   └── styles/        # Global styles
├── public/            # Static assets
└── package.json
```

## Setup

```bash
cd src/frontend
npm install
```

## Running

```bash
npm start
```

## Testing

```bash
npm test
```

## Building

```bash
npm run build
```
